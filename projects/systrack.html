Systrack is designed to monitor and evaluate Service Level Agreement (SLA) performance. It enables users to manually input SLA records, which the system automatically computes and displays in a detailed table format for easy tracking and analysis.

Technologies Used
  • HTML
  • CSS
  • JavaScript
  • PHP
  • MySQL
  • Bootstrap

Core Features

Manual Input of SLA for Automatic Computation
Users can manually enter SLA information such as targets and actual performance data. The system then automatically computes the results to evaluate compliance and efficiency.

SLA Availability Tracking
This feature tracks and records the availability of services based on the defined SLA metrics. It helps users identify performance trends and maintain consistent service quality.

Technologies Used

  • Front-end: HTML, CSS, Bootstrap, Javascript
  • Back-end: PHP, MYSQL
  • Environment: XAMPP (Localhost Server)
